什麼是 CSS？

CSS（Cascading Style Sheets，層疊樣式表）是一種用於描述 HTML 或 XML 文件中元素的外觀和格式的樣式語言。通過使用 CSS，我們可以將內容與樣式分離，從而實現更靈活和可維護的網頁設計。
